# procurocasa.pt

Landing page estática para apresentar o domínio **procurocasa.pt** como disponível para aquisição.

## Publicar com GitHub Pages

1. Crie um repositório no GitHub.
2. Carregue todos os ficheiros deste diretório para a raiz do repositório.
3. Vá a **Settings → Pages**.
4. Em **Build and deployment**, escolha:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Guarde.
6. Em **Custom domain**, confirme `procurocasa.pt`.
7. Configure os DNS do domínio para GitHub Pages.
8. Quando disponível, ative **Enforce HTTPS**.

## Ficheiros incluídos

- `index.html`
- `styles.css`
- `favicon.svg`
- `og-image.svg`
- `robots.txt`
- `sitemap.xml`
- `CNAME`
